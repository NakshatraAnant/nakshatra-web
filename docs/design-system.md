# NAKSHATRA Design System

## Visual Direction

NAKSHATRA should feel:
- cinematic
- emotionally immersive
- premium
- intimate
- emotionally quiet
- futuristic
- elegant
- luxury-tech

The interface should feel emotionally similar to:
- Apple Music at night
- Arc Browser
- Spotify emotional atmosphere
- Pinterest editorial minimalism

Avoid:
- bright gradients
- generic neon
- tacky astrology visuals
- rainbow cosmic effects
- cluttered UI
- dopamine-chaos feeds

---

# Typography

Typography should feel:
- editorial
- luxurious
- cinematic
- emotionally intelligent

Use:
- elegant serif for emotional headings
- clean sans-serif for interface systems

Spacing should feel:
- breathable
- calm
- immersive
- intentional

---

# Motion System

Animations should feel:
- smooth
- atmospheric
- emotionally intelligent
- cinematic

Avoid:
- bouncy animations
- flashy motion
- excessive transitions

Motion should feel:
- subtle
- elegant
- premium

---

# Color Direction

Primary palette:
- deep black
- cosmic navy
- muted violet
- moonlight silver

Accent palette:
- soft glow
- subtle gold
- atmospheric gradients

Avoid:
- oversaturated purple
- rainbow colors
- bright astrology aesthetics

---

# UI Philosophy

The interface should prioritize:
1. Emotional immersion
2. Calm interaction
3. Premium restraint
4. Atmospheric layering
5. Identity expression

The UI should NEVER feel:
- like productivity software
- like a dashboard
- like a crypto app
- like a gaming UI
- like generic SaaS
