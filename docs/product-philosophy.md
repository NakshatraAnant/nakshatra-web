# NAKSHATRA Product Philosophy

NAKSHATRA is an emotional identity social network.

The platform helps users discover:
- emotional compatibility
- emotional belonging
- identity resonance

NAKSHATRA must NEVER feel:
- cheesy spirituality
- astrology-heavy
- generic social media
- productivity SaaS
- dopamine-chaos social apps

The product should feel:
- cinematic
- emotionally immersive
- emotionally quiet
- luxury-tech
- premium
- intimate
- mysterious
- emotionally intelligent

Priority order:
1. Emotional resonance
2. Identity expression
3. Emotional discovery
4. Social belonging
5. Features
